# 아웃소싱 프로젝트 🛵💨
아웃소싱 컨셉의 배달 어플리케이션 구현 프로젝트
회원가입/로그인 및 가게, 메뉴, 주문, 리뷰 CRUD 기능을 제공합니다.

## 🛠️ 개발 환경
**IDE** : IntelliJ <br>
**프레임워크** : Spring Boot 3.4.4 <br>
**데이터베이스** : MySQL / JPA <br>
**빌드 도구** : Gradle-Groovy <br>
**언어** : Java 17 <br>
**의존성** : Lombok / Spring Web / JDBC API / Spring Data JPA / MySQL Driver / Validation / Java Mail Sender <br>
**테스트 도구** : Postman <br>

## 🕺 역할 분담
기능 | 담당
-- | --
회원가입 / 로그인 | 김정연
메뉴 / 가게 CRUD | 이진아
주문 / 리뷰 CRUD | 류형철
부가서비스 / 광고 / 대시보드 | 이형진
소셜 로그인 / 장바구니 / 알림 | 이수빈
테스트코드 및 예외 | 공통사항
